# Car Price Prediction - User Manual

## Overview
This is a machine learning web application that predicts the selling price of a used car based on various features such as kilometers driven, car age, fuel type, seller type, transmission, and ownership history.

The prediction model is trained using a **Random Forest Regressor** and is integrated into a **Flask-based web interface**.
