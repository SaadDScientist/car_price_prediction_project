
Car Price Prediction - User Manual

Overview:
This is a machine learning web application that predicts the selling price of a used car based on various features like kilometers driven, car age, fuel type, seller type, transmission, and ownership history. The prediction model is trained using a Random Forest Regressor and is integrated into a Flask-based web interface.

Requirements:
- Python 3.x
- pip (Python package manager)
- Required Libraries:
  - Flask
  - pandas
  - numpy
  - scikit-learn
  - pickle

To install Flask, run:
pip install flask

How It Works:

1. Model Training (model_training.ipynb)
   - Load and clean the dataset (Car Dekho dataset).
   - Feature engineering is performed: year to age, encoding of categorical variables.
   - Train a Random Forest Regressor model.
   - Save the trained model as 'model.pkl' using pickle.

2. Flask Web App (app.py)
   - Loads the model from 'model.pkl'.
   - Serves the homepage form (index.html) where the user inputs car details.
   - Processes form data and formats it into the required input format for the model.
   - Generates a prediction and displays it along with details on the result page (result.html).

How to Run the Project:

1. Make sure all files are in the correct structure:
   car_price_app/
   ├── app.py
   ├── model.pkl
   ├── model_training.ipynb
   ├── templates/
   │   ├── index.html
   │   └── result.html

2. Open cmd 
3. Navigate to your folder: car_price_app:
	cd "Desktop\car_price_app:"

4. Run the Flask application:
   python app.py

5. Open your browser and go to:
   http://127.0.0.1:5000

6. Fill in the car details and click on "Predict Price" to get the estimated price in PKR.

Notes:
- The prediction is displayed in PKR (Pakistani Rupees).
- The model is trained using cleaned and encoded historical car sale data.
- The UI is styled for clarity and ease of use.

Files Included:
- app.py : Main backend Flask application
- model.pkl : Trained machine learning model
- model_training.ipynb : Jupyter notebook used for training the model
- templates/index.html : Input form for user data
- templates/result.html : Output display page
- user_manual.txt : This manual
